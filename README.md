# App "What Is The News" 📰
➜ <b> What is the News</b> est une application "en français 🇫🇷" sur l’actualité récente, <br/>
elle permet de récupérer l'ensemble des articles, selon un thème précis ou une recherche par mot clé, <br/>
de lire l'article sur le site du créateur de ce contenu tout en restant dans l'application elle-même.

## fonctionnement 
à l'ouverture de <b>What is the News</b>, l'application affiche 2 types d'actualités différentes :
<br /> 
- une première dans la partie logo qui est un carousselle présentant 3 news </br> d'un thème déjà défini qui est la musique 🎸 (non modifiable pour l'instant)
- une seconde dans la liste, présentant dans un premier temps les news Apple, </br> qui peut-être modifiéegrâce au bouton loupe qui se trouve en bas à droite.
